ActividadService

class ActividadService(actividadRepo: IActividadRepository, usuarioRepo: IUsuarioRepository, historialService: HistorialService, val dashboardService: DashboardService)

Gestiona la lógica de negocio relacionada con actividades, usuarios e historial.

Coordina la creación, actualización, asignación y filtrado de tareas y eventos, además de mantener el historial de acciones y la integración con el dashboard.
